Problem Statement
Classify the user inputted review as Positive or Negative.

As we all know that in today’s world the technology is increasing with every second and so the data. A data can be of different types such as text, images etc. Similarly, there is a data which is being collected every second in response to another set of data, generally known as reviews or comments. As we can witness a lot of data collection in form of reviews and comments in response to a dataset which can be anything YouTube videos, movies, hotels and so on. So these reviews or comments data should be analyzed and classified in order to evaluate the different aspects such as ratings, popularity and so on depending on the dataset.

Therefore, our team is planning to build a machine learning project working on the principle of NLP (Natural Language Processing) which will be analyzing and predicting the reviews as “Positive” or “Negative”. The dataset used for model building is an review dataset which was obtained from Kaggle platform. We are using Flask to build an API which will be deployed in Heroku platform along with this model from where we will be able to predict the polarity of the user review.
